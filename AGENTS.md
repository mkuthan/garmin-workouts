# AGENTS.md

## Project Overview

This project is a Python package that provides tools for working with Garmin workouts. It includes functionality for parsing, analyzing, and manipulating workout data from Garmin devices. The package is designed to be used in command-line applications, allowing users to manage their Garmin Connect workouts efficiently.

## Setup Commands

- Install dependencies: `uv sync`
- Run all checks (lint, format, type, test): `mise run check`
- Run tests: `mise run test`
- Run linter: `mise run lint`
- Format code: `mise run format`
